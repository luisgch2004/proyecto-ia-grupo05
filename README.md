# 🤖 Proyecto Final de Inteligencia Artificial

¡Bienvenido al repositorio de nuestro proyecto grupal! En este espacio se consolidan todos los desarrollos técnicos, informes analíticos y material audiovisual generados a lo largo del curso. El proyecto está dividido principalmente en dos grandes componentes: el **Bloque A** (Algoritmos clásicos de búsqueda y juegos en Java) y el **Bloque B** (Análisis de datos y modelado de Machine Learning).

---

## 📁 Estructura del Repositorio

A continuación se detalla la organización de las carpetas del proyecto para facilitar la navegación y evaluación:

```text
